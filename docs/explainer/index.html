The solution — layer 1

An open file standard

Services publish /.well-known/safety.md listing their payment addresses.
Machine-readable. Crawlable. Verifiable.
--- version: "1.0" name: "b1e55ed" addresses: - address: "0xB1e55EdD..." chain: "base" label: "Payment address" identity: erc8004_agent_id: 28362 trust: domain: "oracle.b1e55ed.com" ---
✓
Human + machine readable
YAML frontmatter in a Markdown file. Works everywhere.
✓
Zero friction to adopt
Drop one file at a well-known path. Done.
✓
Crawled and indexed daily
safety.md discovers and verifies new services automatically.
